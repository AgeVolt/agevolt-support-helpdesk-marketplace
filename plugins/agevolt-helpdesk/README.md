# AgeVolt Helpdesk

Codex plugin pre AgeVolt support a helpdesk operativu.

Obsahuje:

- skill `agevolt-helpdesk`,
- MCP konfiguraciu pre `https://av-agent.agevolt.com/mcp/helpdesk`,
- runbooky pre WiseCloud touchpoint operacie, OCPP log export a touchpoint remote start/stop nabijania vratane OCPP free charging, touchpoint-local free charging a fallback startu pre neplatene nabijanie a ClickUp komentara + ClickUp task Email mode follow-upu.

Plugin neobsahuje ziadne hesla, tokeny ani zakaznicke exporty. Zive operacie idu cez server-side AgeVolt Helpdesk MCP.
