# AgeVolt Support Helpdesk Marketplace

Marketplace pre support, helpdesk a zakaznicku operativu.

Ma sluzit ludom, ktori riesia poziadavky zakaznikov, incidenty, odpovede, eskalacie, hladanie v support znalostiach a denne alebo tyzdenne support reporty.

Tento marketplace moze neskor obsahovat pluginy pre ticket triage, draft odpovedi, incident summary, WiseCloud diagnostiku, support knowledge a eskalacie na produkt alebo vyvoj.

Aktualny stav: pripravene ako prazdny marketplace, bez pluginov.
