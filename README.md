# AgeVolt Support Helpdesk Marketplace

Marketplace pre support, helpdesk a zakaznicku operativu.

Ma sluzit ludom, ktori riesia poziadavky zakaznikov, incidenty, odpovede, eskalacie, hladanie v support znalostiach a denne alebo tyzdenne support reporty.

Aktualne obsahuje plugin `agevolt-helpdesk` pre helpdesk postupy, WiseCloud touchpoint operacie, OCPP log export, bezpecny remote start/stop nabijania na touchpointe cez AgeVolt Helpdesk MCP, OCPP free charging alebo fallback start pre neplatene nabijanie a ClickUp komentar + ClickUp task Email mode follow-up po operacii.

Pluginy v tomto marketplace su urcene pre interne support pouzitie a nesmu obsahovat tajne hodnoty, realne hesla, tokeny ani zakaznicke exporty.
